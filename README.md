# 🚀 AI Resume Analyzer & Job Match System

An intelligent **AI-powered Resume Analyzer** that evaluates resumes, calculates ATS scores, matches them with job descriptions, and identifies skill gaps using **Machine Learning and NLP**.

---

## 📌 Features

* 📄 Resume parsing and data extraction
* 📊 ATS (Applicant Tracking System) score calculation
* 🎯 Job description matching
* 📉 Skill gap analysis
* 🤖 AI-powered insights using LLMs
* 📈 Interactive dashboard with Streamlit

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Libraries & Tools:**

  * Streamlit
  * Pandas, NumPy
  * Scikit-learn
  * NLP techniques
* **APIs:** Groq API (LLM integration)
* **Frontend:** Streamlit UI

---

## ⚙️ Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/Resume-analyzer.git
cd Resume-analyzer
```

### 2️⃣ Create virtual environment

```bash
python -m venv venv
venv\Scripts\activate
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

---

## 🔐 Environment Variables

Create a `.env` file in the root directory:

```bash
GROQ_API_KEY=your_api_key_here
```

⚠️ Do NOT share your API key publicly.

---

## ▶️ Run the Application

```bash
streamlit run app.py
```

---

## 📊 How It Works

1. Upload your resume
2. System extracts text using NLP
3. Calculates ATS score
4. Matches with job description
5. Displays skill gaps and improvement suggestions
